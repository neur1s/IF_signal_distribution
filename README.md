## Requirements

This notebook was tested with **Python 3.11.9 and higher** and the following packages:

- `numpy >= 1.24`
- `matplotlib >= 3.7`
- `tifffile >= 2023.4`

To install the required packages using **conda**:

```bash
conda install python=3.11.9 numpy>=1.24 matplotlib>=3.7 tifffile>=2023.4
